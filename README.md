# HR Analytics Project – Employee Attrition Analysis

## Overview

This project analyzes employee attrition data to identify factors influencing employee turnover. Using Python, Pandas, Matplotlib, and Seaborn, the project performs exploratory data analysis and visualization to uncover trends related to employee satisfaction, salary, projects, and department-wise attrition.

## Objectives

* Analyze employee attrition patterns.
* Identify departments with high employee turnover.
* Study the relationship between projects and attrition.
* Examine salary distribution and its impact on attrition.
* Visualize correlations among employee-related features.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Key Findings

* Total Employees: 14,999
* Employees Left: 3,571
* Employees with fewer than 3 projects showed significantly higher attrition.
* Sales, Technical, and Support departments experienced the highest employee turnover.
* Lower salary levels were associated with higher attrition rates.

## Visualizations

* Department-wise Attrition Analysis
* Projects vs Time at Company
* Salary Distribution vs Attrition
* Correlation Heatmap

## Conclusion

The analysis highlights major factors contributing to employee attrition and provides insights that can help organizations improve employee retention strategies.
